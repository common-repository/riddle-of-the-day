== Description ==
Displays a random riddle of the day in your sidebar, to get answer click the answer button.


== Installation ==

1. Upload `riddles.php` to the `/wp-content/plugins/` directory
2. Go to plugins section in your dashboard and activate the plugin
3. After activation find the Appearance tab, and locate the link called Widgets and select that
4. In the section labled Available Widgets locate the one labled "Riddles". 
5. Just drag it to whatever bar you would like the plugin on and type in your title or go with the default option.
